## 下载并安装Redis
1. 打开链接 https://github.com/redis-windows/redis-windows/releases 下载Redis
2. ![](assets/使用教程/file-20251211145300146.png) 并将下载的 Redis-8.4.0-Windows-x64-cygwin-with-Service.zip 解压，注意建议不要解压到中文目录下
3. ![](assets/使用教程/file-20251211145430666.png) 运行解压后的 start.bat 文件![](assets/使用教程/file-20251211145458057.png) 此时出现 Redis 运行窗口，不要关闭，关闭以后Redis即被关闭。
## 下载并安装IDEA
1. 打开链接 https://www.jetbrains.com/zh-cn/idea/download/?section=windows 进行下载![](assets/使用教程/file-20251211145651337.png) 打开IDEA安装程序![](assets/使用教程/file-20251211150159635.png) ![](assets/使用教程/file-20251211150227714.png) 除了需要勾选创建桌面快捷方式，其他一直点击下一步即可。
## 下载安装NodeJS

2. ![](assets/使用教程/file-20251211150620309.png) 打开下载好的nodejs安装程序![](assets/使用教程/file-20251211150704392.png) ![](assets/使用教程/file-20251211150714967.png) ![](assets/使用教程/file-20251211150727363.png) ![](assets/使用教程/file-20251211150739455.png) ![](assets/使用教程/file-20251211150747550.png) ![](assets/使用教程/file-20251211150755441.png) 等待即可安装完成
# 运行项目
## 导入项目
1. 打开IDEA![](assets/使用教程/file-20251211150941110.png) 点击打开按钮 选择之前解压的代码目录下的pom.xml文件![](assets/使用教程/file-20251211151046612.png) ![](assets/使用教程/file-20251211151054890.png) ![](assets/使用教程/file-20251211151326122.png)  
2. 导入数据库初始数据![](assets/使用教程/file-20251211151434499.png) 在找到 application.yml 点击从现有属性创建数据源按钮 ![](assets/使用教程/file-20251211151555721.png) 将数据库输入框中的内容全部删除 在点击确定![](assets/使用教程/file-20251211151654413.png) 找到 scheme.sql 右键点击运行‘scheme.sql’![](assets/使用教程/file-20251211151743173.png) 添加目标数据源![](assets/使用教程/file-20251211151811493.png) 再点击添加脚本![](assets/使用教程/file-20251211151839761.png) 将之前解压的代码文件目录下的 data.sql 文件打开![](assets/使用教程/file-20251211151906355.png) 点击运行 数据库初始数据即导入成功
3. 运行后端项目![](assets/使用教程/file-20251211152006311.png) ![](assets/使用教程/file-20251211152018639.png) 如有该弹窗，点击启用注解处理
4. 运行前端项目![](assets/使用教程/file-20251211152055181.png) ![](assets/使用教程/file-20251211152114633.png)  ![](assets/使用教程/file-20251211152131035.png)![](assets/使用教程/file-20251211152146036.png) 选择之前解压代码目录下的frontend目录下的package.json文件![](assets/使用教程/file-20251211152221710.png) 脚本选择dev![](assets/使用教程/file-20251211152306794.png) 点击确定![](assets/使用教程/file-20251211152351557.png) 右键项目目录下的frontend文件夹点击打开于-终端![](assets/使用教程/file-20251211152431679.png) 在下方终端当中 输入命令 npm  install 按下回车 等待一段时间![](assets/使用教程/file-20251211152507657.png) 这时候就好了![](assets/使用教程/file-20251211152538557.png) 在右上角找到dev 点击运行![](assets/使用教程/file-20251211152552644.png) 出现此窗口前端运行成功
5. 浏览器访问 http://localhost:5173 即访问系统
