# USER
基础用户表

| 数据类型    | 字段        | 约束     | 备注    |
| ------- | --------- | ------ | ----- |
| varchar | id        | PK     | 用户ID  |
| varchar | username  | unique | 用户名   |
| varchar | password  |        | 密码    |
| varchar | nickname  |        | 昵称    |
| varchar | real_name |        | 真实姓名  |
| varchar | email     | unique | 邮箱    |
| varchar | avatar    |        | 头像URL |

# ROLE
角色表

| 数据类型    | 字段   | 约束     | 备注   |
| ------- | ---- | ------ | ---- |
| varchar | id   | PK     | 角色ID |
| varchar | name | unique | 角色名  |
# USER_ROLE
用户角色关联表


| 数据类型    | 字段      | 约束       | 备注      |
| ------- | ------- | -------- | ------- |
| bigint  | id      | PK       | 主键ID、自增 |
| varchar | user_id | FK（逻辑外键） | 用户ID    |
| varchar | role_id | FK（逻辑外键） | 角色ID    |
# ROLE_PERMISSION
角色权限关联表

| 数据类型    | 字段         | 约束       | 备注      |
| ------- | ---------- | -------- | ------- |
| bigint  | id         | PK       | 主键ID、自增 |
| varchar | role_id    | FK（逻辑外键） | 角色ID    |
| varchar | permission |          | 权限      |

# COUNSELOR
咨询师表

| 数据类型    | 字段           | 约束       | 备注         |
| ------- | ------------ | -------- | ---------- |
| varchar | id           | PK       | 咨询师ID      |
| varchar | user_id      | FK（逻辑外键） | 对应用户ID     |
| varchar | name         |          | 姓名         |
| varchar | specialty    |          | 擅长领域       |
| text    | introduction |          | 个人简介       |
| decimal | rating       |          | 评分（保留一位小数） |
# APPOINTMENT
预约表

| 数据类型     | 字段               | 约束       | 备注                         |
| -------- | ---------------- | -------- | -------------------------- |
| bigint   | id               | PK       | 主键ID、自增                    |
| varchar  | counselor_id     | FK（逻辑外键） | 咨询师对应用户ID                  |
| varchar  | student_id       | FK（逻辑外键） | 学生对应用户ID                   |
| datetime | appointment_time |          | 预约时间                       |
| tinyint  | type             |          | 预约类型（1：线上；2：线下）            |
| tinyint  | status           |          | 状态（1：待确认；2：未完成；3：已完成；4：爽约） |
| tinyint  | rating           |          | 评分（1-5）                    |


# TREEHOLE_POST
树洞帖子表

| 数据类型      | 字段        | 约束       | 备注                  |
| --------- | --------- | -------- | ------------------- |
| varchar   | id        | PK       | 帖子ID                |
| varchar   | title     |          | 标题                  |
| longtext  | content   |          | 内容                  |
| varchar   | user_id   | FK（逻辑外键） | 发帖人用户ID             |
| timestamp | post_time |          | 发帖时间                |
| tinyint   | pass      |          | 审核是否通过（0：不通过； 1：通过） |

| --------- | ---------- | -------- | --------- |
